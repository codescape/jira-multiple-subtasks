### Will there be a Jira Cloud version of Multiple Subtasks for Jira?

At the moment only Jira Server and Jira Data Center compatible versions of Multiple Subtasks for Jira are available.
We have already started to develop a Jira Cloud version of Multiple Subtasks for Jira.
A first release will be announced as soon as it is available.

### Where does Multiple Subtasks for Jira persist data?

Multiple Subtasks for Jira does not persist any data outside your instance.
All data persisted by Multiple Subtasks for Jira can be found in the following database tables:

* `AO_######_SUBTASK_CONFIG`
* `AO_######_SUBTASK_TEMPLATE`
